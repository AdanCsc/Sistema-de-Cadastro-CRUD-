# Sistema de Cadastro em Python

Projeto CRUD desenvolvido em Python para gerenciamento de usuários via terminal.

## 🚀 Funcionalidades
- Cadastro de usuários
- Listagem de dados
- Edição de informações
- Exclusão de registros

## 🧠 Tecnologias
- Python

## ▶️ Como executar
```bash
python cadastro.py

```md
